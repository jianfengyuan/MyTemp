## 10. 正则表达式匹配

### 题目描述

> 给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '*' 的正则表达式匹配。
>
> '.' 匹配任意单个字符
> '*' 匹配零个或多个前面的那一个元素
> 所谓匹配，是要涵盖 整个 字符串 s的，而不是部分字符串。

### 代码

```python
class Solution:
    def isMatch(self, s: str, p: str) -> bool:
        m = len(s)
        n = len(p)
        def matches(i, j):
            if i==0:
                return
            if p[j-1] == ".":
                return True
            return s[i-1] == p[j-1]
        dp = [[False for _ in range(n+1)] for _ in range(m+1)]
        dp[0][0] = True
        for i in range(len(dp)):
            for j in range(1,len(dp[i])):
                if p[j-1] == "*":
                    dp[i][j] |= dp[i][j-2]
                    if matches(i, j-1):
                        dp[i][j] |= dp[i-1][j]
                else:
                    if matches(i, j):
                        dp[i][j] |= dp[i-1][j-1]
        return dp[-1][-1]
```
