## 680. 验证回文子串

### 题目

> 给定一个非空字符串 `s`，**最多**删除一个字符。判断是否能成为回文字符串。

### 分析

> 贪心算法, 假设字符串边界为`i, j`
>
> 如果两边的字符相等, 继续向中间贪心计算
>
> 如果两边字符不等, 尝试`s[i...j-1] or s[i+1...j]`, 因为仅有一次删除机会

### 代码

```python
class Solution:
    def validPalindrome(self, s: str) -> bool:
        def is_valid(s, i, j):
            while i <= j:
                if s[i]!= s[j]:
                    return False
                i += 1
                j -= 1
            return True
        left, right = 0, len(s) - 1
        while left <= right:
            if s[left] == s[right]:
                left += 1
                right -= 1
            else:
                return is_valid(s, left+1, right) or is_valid(s, left, right-1)
        return True
```
