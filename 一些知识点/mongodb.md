## MongoDB入门

| SQL术语     | MongoDB术语 | 解释                                     |
| ----------- | ----------- | ---------------------------------------- |
| database    | database    | 数据库                                   |
| table       | collection  | 表/集合                                  |
| row         | document    | 数据记录/文件                            |
| column      | field       | 字段/域                                  |
| index       | index       | 索引                                     |
| primary key | primary key | 主键, MongoDB自动在每个集合中添加_id主键 |
